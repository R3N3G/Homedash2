HomeDash

HomeDash is a simple dashboard that allows to monitor and interact with many different services in order to have a single entry point for one's dedicated server via a set of Plugins. It's easy to install, setup and use.

Customisable

The dashboard plugins can be arranged and resized in any way. The mobile, tablet, desktop layouts and your custom ones have their own arrangement so each screen is really tailor-made.

Responsive

HomeDash is fully responsive, it will adapt to any device, from mobile to desktop. Custom sizes of layouts can be created so it can really fit any device devices. A kiosk mode is also available so it can be displayed full screen on TVs or Raspberry Pis with small screens.

Organisable

HomeDash allows to have multiple boards so plugins can be categorized to avoid having a cluttered dashboard.

All in one

Homedash can import plugins from another instance to keep everything at one place.

Plugins

Homedash has many different plugins to connect and interact with many services.

System Info

Monitor the CPU and Ram of the server hosting HomeDash.

Hard Drive

Monitor a hard drive size.

Full screen: Browse, create folders, upload, copy/cut paste files throughout the selected drive.

Network Monitor

Monitor a network interface of the server hosting HomeDash

Transmission

Monitor Download and Upload speed of a Transmission instance. Throttle speed and add torrents easily.

Full screen: Manage all torrents of the instance.

CouchPotato

Easily add movies to a CouchPotato instance wanted list.

PiHole

Monitor the ad-blocking software PiHole

Docker

Shows the amount of running containers

Full screen: Manage containers and images

Docker Compose

Shows the amount of running containers of a docker compose stack

Full screen: Show all the containers details, edit the docker-compose file and run docker-compose commands

Dynamic DNS

Refreshes a dynamic DNS IP whenever the server's IP changes. Dynamic DNS providers supported: OVH, no-ip.org, dyn.com

Logs

Shows the level of activity on a log file

Full screen: Follows the log file content

Spotify

Shows the currently playing song on a Spotify Account

Plex

Displays currently playing Videos on a Plex server

Sonarr

Displays the upcoming TV shows from a Sonarr installation and add new TV show in full screen mode

MMA

Get updated on the upcoming Mixed Martial Arts events and browse fighter records
